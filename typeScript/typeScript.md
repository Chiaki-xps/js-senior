## 1. 基本类型

```typescript
// 1.基本数据类型（Tip：Symbol是基本数据类型的一种）
let str: string = 'string';
let num: number = 1;
let bool: boolean = true;
let u: undefined = undefined;
let n: null = null;
let obj: object = { name: 'chiaki' };
let big: bigint = 100n;
let sym: symbol = Symbol('me');
```
