```js
for (var y = 0; y <= 2; y++) {

    (function (y) {

        setTimeout(function () {

            console.log(y);

        }, y * 1000);

    })(y);

}
// 函数拥有块级作用域，每次立即执行，保存当前传入的y 所以0 1 2。setTimeout想上找找到的就是函数里的y
```

```js

```

```js

// setTimeout会执行传入的回调函数，类似与(functon(){})()再加上(),结果成了console.log(y)()
```
